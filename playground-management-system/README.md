# 游乐场管理系统

基于 React + Ant Design Pro 开发的游乐场管理系统原型。

## 项目概述

这是一个为游乐场/主题乐园设计的综合管理系统，包含营业管理、会员管理、商品管理、营销推广、设备管理等多个模块。

## 技术栈

- **前端框架**: React 18
- **UI 组件库**: Ant Design 5.x + Ant Design Pro Components
- **构建工具**: UmiJS 4.x
- **图表库**: Recharts
- **语言**: TypeScript

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

构建后的文件位于 `dist` 目录，可以部署到任何静态托管服务：

- 腾讯云开发（CloudBase）
- 阿里云 OSS
- GitHub Pages
- Vercel
- Netlify
